# Info-Modelling-project

## Objects

Norwegian foods

#### Schema

-   id: number (auto-inc from 1)
-   name: string
-   image: string (link)
-   description: string
-   source (url): string
-   Our rating: number

### How to run

```
cd server
npm install
npm start
```

Open a new terminal

```

cd norwegian-foods
npm install
npm run dev
```

#### Alternatively

In the root folder:

```
npm run dev
```
